Менеджер задач

Это веб-приложение — простой менеджер повседневных задач. Оно позволяет добавлять, отмечать как выполненные и удалять задачи. Проект разработан в рамках изучения основ веб-разработки с использованием HTML, CSS и JavaScript

Приложение позволяет:

  • Добавлять новые задачи через модальное окно на основе тега dialog;
  • Отмечать задачи как выполненные;
  • Удалять ненужные задачи;
  • Работать с интерфейсом на любых устройствах — от компьютеров до смартфонов.

Технологии: HTML5, CSS3 (Flexbox, медиазапросы), JavaScript (для управления диалоговыми окнами и бургер-меню).

Проект разработан как учебное задание по дисциплине «Основы веб-разработки».
