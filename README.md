# Tasks Manager (Лабораторная работа)

Веб-интерфейс для отображения списка задач в виде таблицы. Проект разработан в рамках лабораторной работы по дисциплине "Основы веб-разработки".

## О проекте
Сайт позвояет добавлять задачи, отмечать их как выполненные и удалять при необходимости. Также есть возможность добавлять сроки выполнения к задачам,
экспортировать их в формате ICS и отмечать их на встроенном календаре.

## Технологии
Проект реализован на классическом стеке фронтенда без использования серверной части:
- **HTML5** — разметка страницы и таблицы.
- **CSS3** — стилизация и оформление.
- **JavaScript** — логика работы.
- **Node.js** — сборка проекта

## Как запустить
1. Установите [Node.js](https://nodejs.org/)
2. Клонируйте репозиторий
3. Выполните `npm install`
4. Запустите `npm start`
